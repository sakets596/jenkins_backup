Specify the conditions under which the sound will be played. A condition is defined as a transition from the job's previous result to this one. Muliple mappings can be added to cover all combinations.
  • On build result
    • Select the result of a build that this sound applies to.
  • For previous build result
    • Select the previous results to define the transition - at least one must be selected.
      • NB - Not Built
      • Ab - Aborted
      • Fa - Failed
      • Un - Unstable
      • Su - Succeeded
  • Play sound
    • Choose a sound to play.
The sound will be played according to the global System Configuration settings.
